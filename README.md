Mi meta final es hacer una APP para Android estilo EL SISTEMA de Solo Leveling, con una interfaz que sea clavada a la del anime o el manhwa, para ir progresando en la vida real tal cual como en la serie. Quiero una herramienta visual y potente que me permita gamificar mi disciplina y mis metas diarias.

Quiero que el sistema tenga estos stats:

  -  Fuerza (STR): Se sube con el entrenamiento físico de fuerza y gimnasio.

  -  Inteligencia (INT): Se gana mediante el estudio técnico (como programación), formación teórica y lectura.

  -  Destreza (DEX): Para tareas que requieran precisión, habilidad manual o coordinación.

  -  Voluntad (WILL): El stat clave. Se gana cumpliendo el déficit alimenticio, manteniendo horarios estrictos y terminando tareas pesadas sin excusas.

  -  Fondo (END): Para la actividad cardiovascular, caminatas largas y resistencia física diaria.

  -  Enfoque (FOC): Stat principal ahora mismo. Se sube con sesiones de trabajo profundo y lectura concentrada sin distracciones.

  -  Social (SOC) y Presencia (PRES): Para mejorar la capacidad de networking, la seguridad personal y la proyección ante los demás.

  -  Adaptabilidad (ADP): Un stat de resiliencia inspirado en mecánicas de RPG. Se gana gestionando imprevistos, cambios de rutina o manteniendo la disciplina en momentos complicados bajo la filosofía de: "ES LO QUE HAY".

Quiero que cada stat tenga su propio nivel con costes de experiencia exponenciales. Además, habrá un Nivel General basado en la EXP total acumulada. Esto marcará mi RANGO, que irá desde el E (nivel 0-5) para principiantes, pasando por el D (nivel 5-15) de Cazador, el C (nivel 15-30), hasta llegar al Rango S o SS+ de MONARCA (nivel 100+).

Para fomentar la constancia, quiero un sistema de racha con un multiplicador de experiencia acumulativo. Pero si fallo en las misiones críticas, habrá CASTIGOS obligatorios:

  -  Si no se cumple el entrenamiento o el estudio: Privación de ocio y acceso al PC.

  -  Si no se cumple la alimentación: Compensación con actividad física extra.

  -  Como misión fija semanal: Sesiones de lectura obligatorias para potenciar INT, FOC y WILL.

El sistema debe incluir opciones de exportar/importar progreso y guardado automático en el navegador. En el futuro, quiero que sea una APP nativa con login de usuario y guardado en la nube, manteniendo siempre la estética de la serie.

De momento estoy desarrollando la base en HTML, CSS y JS con ayuda de IA. Estoy echándole ganas, cumpliendo objetivos y estudiando a tope. Este sistema es la herramienta definitiva para llevar mi disciplina al siguiente nivel.
